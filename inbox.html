Introduction to Css

Css (cascading stylesheet) this is a styling language that is used to construct the vision of the web page

way to write css

  1. inline css
  2. internal css
  3. external css

Inline CSS is the technique to define the single element with the insert style sheets in an HTML document.

An internal CSS is used to define within a style tags inside the haed section of html element

External CSS is a form of CSS which is used to add styling to multiple HTML pages at a time
